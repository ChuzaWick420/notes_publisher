# Mathematical Notations
This document covers some of the mathematical notations used in other parts of my notes.

## Because
### Symbol

$$\because$$

### Usage
"Because one added to two is three"  

$$\because 1 + 2 = 3$$

## Therefore
### Symbol

$$\therefore$$

### Usage
"Therefore, $a$ is equal to $b$"  

$$\therefore a = b$$
