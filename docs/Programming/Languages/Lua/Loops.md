# Loops

## For

```lua
for i = 0, 20, 2 do
	print(i)
end
```

Notice how there are `3` values.

1. Starting value
2. Ending value
3. Increment or decrement by

## While

```lua
while num < 50 do
	print("hi")
	num = num + 1
end
```

`block scopes` usually end with `end` keyword.

## Another Variant

```lua
repeat
	print("hi")
until n < 20
```
