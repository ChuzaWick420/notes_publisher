# Formal Definition
For any number $\epsilon > 0$ , if we can find an interval $(x_0, x_1)$ such that  

$$L - \epsilon < f(x) < L + \epsilon$$

except possibly $x = a$ then,  

$$\lim_{x \rightarrow a} f(x) = L$$

Therefore, $f(x)$ exists in the interval $(L - \epsilon, L + \epsilon)$
