# What is Computing

<span style="color: gray;">Dated: 05-12-2025</span>

Computing is the process to complete a given goal-oriented task by using computer technology.

Computing may include the design and development of software and hardware systems for broad range of purposes.

Used for structuring, processing and managing any kind of information - to aid in the pursuit of scientific studies and making intelligent systems.

## History

1. Batch Era: Execution of series of programs on a computer without manual intervention.
2. Time Sharing: Sharing of computing resource among many users by means of multiprogramming and multi-tasking.
3. Desktop Era: A personal computer provides computing power to one user.
4. Network Era: Systems with shared memory and distributed memory.
