# Escape Sequences

## Single Quote

There is only one.

```yaml
key: 'have ''one'' escape pattern'
```

## Double Quote

There are many like `\n`, `\"` etc.

## Unicode

The `UTF-8`, `16` and `32` need to be encoded

```yaml
key: \u00B2
```

## Special Characters

Characters like `$`, `&` etc must be enclosed in quotes.
